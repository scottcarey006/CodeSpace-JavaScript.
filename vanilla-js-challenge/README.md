# Rock Paper Scissors (JavaScript Project)

This is a beginner JavaScript project where I built a simple Rock, Paper, Scissors game.

The user selects an option using buttons, and the computer randomly selects its choice. The result is then displayed on the page.

---

## ⚙️ How It Works

1. The user clicks a button (rock, paper, or scissors)
2. JavaScript detects the click using an event listener
3. A random choice is generated for the computer
4. The choices are compared using if statements
5. The result is displayed on the page

---

## 📁 Project Structure

- `index.html` → page structure
- `style.css` → basic styling
- `script.js` → game logic
